# go-tfl

A golang client for using the [Transport for London Unified API](https://api.tfl.gov.uk/)

## completion

The sections currently implemented are:
- [ ] Accident Stats
- [ ] Air Quality
- [x] Bike Point
- [ ] Cycle Superhighway
- [ ] Journey
- [ ] Line
- [ ] Mode
- [ ] Notification
- [ ] Occupancy
- [ ] Place
- [ ] Road
- [ ] Search
- [ ] Stop Point
